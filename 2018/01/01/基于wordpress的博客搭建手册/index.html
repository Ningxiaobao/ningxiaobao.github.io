个人博客搭建

第一步：

申请服务器及域名

1.你需要一个服务器和域名（以腾讯云为例）

2.申请域名：

进入腾讯云首页（console.qcloud.com），点击云产品 –> 云服务器 –>选择新建一台云主机（系统选用：ubuntu），并按照自己的需要设置

创建完成后，系统会自动分配给你相应的公网 IP 和内网 IP 例：123.206.xx.xx [公]10.141.xx.xx[内]）

3.域名申请：

返回腾讯云首页，点击云产品 –> 域名管理 –>域名注册（注册的新域名会再提交的一两天后，得到腾讯的认证）

域名注册成功后，点击你的域名右方的 “解析” ，将记录类型的 A 的记录值，改为你的公网 IP.

服务器和域名这两个前提条件解决后，面对这个“崭新”的服务器，是时候给他配置装备了

第二步

安装及配置 PHP、apache 和 MYSQL 的环境

附：此部分可以忽略，但小宝建议你还是装上 本部分转自 [一个你非常值得看的博客]

此处介绍 OhMyzsh

ZSH 是一个 Unix Shell ，它在兼容标准的 POSIX （对于小宝这种小白来说，它最吸引我的功能就是命令补全错误纠正、简写和漂亮的界面主题）

Zsh 很好很强大,但是由于其配置起来过于繁琐复杂,严重的影响使用效率，所以 OhMyzsh 应用而生

OhMyzsh 的好处太多了，小宝就不一一列举了，直接上安装手法

附1.安装 OhMyzsh 前，必须安装 Zsh、curl、git 等 [别问小宝为什么，因为这是最简单的方法，如果你想深入快去看上面那个博客]

附2.首先安装 Zsh,打开 terminal 命令：sudo apt-get install zsh [这可是必须的，小宝第一次就直接装了OhMyzsh 结果呢 自然是被嘲讽了]

附3.接着安装 curl, 命令 sudo apt-get install curl

附4.然后安装 git, 命令 sudo apt-get install git

附5.最后安装 ohmyzsh, 命令: curl -L http://install.ohmyz.sh | sh [终于可以安装了]

附6.终于安装完成了 T_T .赶快切换默认 shell 为 zsh : chsh -s /bin/zsh

附7.看到了 linux 的命令行，瞬间漂亮了有么有！！ 再告诉你默认主题是 robbyrussell ，可这不是唯一选择哦

主要参考网站：LAMP的安装教程

使用 Xshell5 等软件，远程登陆自己的服务器。

1.对服务器的软件源进行功能更新：sudo apt-get update [更新软件源]

2.对服务器的系统软件进行功能升级：sudo apt-get upgrade [更新所有的软件]

小小解析一下：

apt-get update ： 访问源列表里的每个网址，并读取软件列表，然后保存在本地电脑

apt-get upgrade ： 会把本地已安装的软件，与刚下载的软件列表里对应软件进行对比，如果发现已安装的软件版本太低，就会更新该软件

3.安装 apache

命令：sudo apt-get install apache2 （附：一路 Y 下去就好）

安装完后，可通过访问自己的公网 IP ，若出现 Apache2 Ubuntu Default Page 页面，证明 apache 安装完成

4.安装 MYSQL

命令： sudo apt-get install mysql-server php5-mysql

命令：sudo mysql_secure_installation 可以提高数据库的安全性

出现 MYSQL 的安装界面，按提示输入 root密码（并确认密码）；

Change the root password 选择 n [是否修改 MYSQL 的 root 用户的密码]

Remove anonymous users 选择 n [是否移除匿名用户]

Disallow root login remotely 选择 n [不允许 root 用户的远程登陆数据库]

Remove test database and access to it 选择 n [移除初始的test数据库（其实都行）]

Reload privilege tables now 选择 Y [现在重新加载权限表] （注意哦，这个是 Y）

5.安装 PHP

命令： sudo apt-get install php5 libapache2-mod-php5 php5-mcrypt （附： Y 下去就好）

安装完成后，修改 apache 的配置文件，使 apache 执行文件的优先级改变（即:提高 index.php 的优先级）

命令 ： sudo vim /etc/apache2/mods-enabled/dir.conf

修改前：

<IfModule mod_dir.c>

</IfModule>

修改后：

<IfModule mod_dir.c>

</IfModule>

修改完成后,保存文件 :wq (保存修改并推出), PHP 的安装大体完成，

若你想使用 PHP 的命令行，使用命令： sudo apt-get install php5-cli

检测 PHP 安装是否正确

编写 index.php 文件（附：检测万要记得删，否则回合以后重名）

内容为：<?php phpinfo();?>

访问自己的公网 IP，显示 php 首页，说明安装正常。

第三步

安装 wordpress

1.首先从 wordpress 的中文官网,下载最新版的 wordpress 4.5.3

2.利用 FileZilla Client 等软件，将 wordpress 4.5.3 的压缩包上传到服务器

3.我的天,这个压缩包是 ZIP 的, tar 命令不能接zip的压缩包，那怎么解压缩呢？ 没办法，只能安装新工具 zip

命令 ：sudo apt-get install zip

安装完成后 ：使用命令 ：unzip -b wordpress4.5.3.zip 解压到 /var/www/html 目录下

附：小宝对 zip 的命令不是很熟，有兴趣的小伙伴可以自己去看 zip/unzip 的参数及作用，这里我不多说了

解压完成后，

第四步

设置 wordpress 的相关条件

1.使用 MYSQL 建立一个 名字为 wordpress 的数据库（别名也可以，但小宝不建议你这么做）

命令 ： mysql -u root -p 输入用户密码

建立数据库 ： create database wordpress

2.建立完成后，在 wordpress 的注册界面，按提示填写你的信息，将自动生出数据库信息

3.将生成的信息写入新建的 wp_config.php 替换 wp_config.example.php,接着点击“立即安装”,此时你的个人博客已经搭好了

4.再次访问你的公网 IP，是不是有一点小激动，第一次我也是这样（害羞），

5.进入设置，常规选项修改 wordpress 的 URL 和站点地址 URL 为你的域名，固定连接选项修改固定链接设置为朴素

6.最后再说一点吧，将下载的 wordpress 解压到 theme 文件夹中，就可以应用这个主题了哦
